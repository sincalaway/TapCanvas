# 章节分镜工作台 ASCII

目标：用一个固定布局把“资产库、片段脚本编辑、结果预览、分镜条、章节控制区”锁死，避免实现时退回通用后台布局。

相关文档：

- [固定流水线 Checklist](./CHAPTER_STORYBOARD_WORKSPACE_FIXED_PIPELINE_CHECKLIST.md)

```text
└────────────────────────────────────────────────────────────────────────────────────────────┘
```

说明：

- `漫剧*` / `章节工作台` 是当前项目内打开的**全屏弹窗层**
- 不是新页面壳，不是脱离项目的独立系统
- 不是画布上的浮动面板
- 打开后覆盖当前项目页面，关闭后回到原画布视图

## 3. 当前镜头切换后的联动

```text
点击镜头卡
  -> 中间片段脚本区切到该镜头所属片段
  -> 右侧预览/详情切到该镜头
  -> 左侧资产栏高亮该镜头引用的角色/场景/道具/风格
  -> 若需要查看原文，从“章节原文证据抽屉”展开
  -> 若该镜头在画布有对应节点，可显示 [定位节点]
```

## 5. 中间主区与正文证据的关系

```text
默认主位
  -> 当前片段脚本编辑区
  -> 可直接编辑片段脚本
  -> 可直接用 @ 引资产

章节全文
  -> 不是默认主位
  -> 作为章节证据源按需展开
  -> 用于验证镜头是否贴合原文，不占主编辑面
```

## 7. 与当前系统的真实关系

```text
工作台是主交互层
  -> 用户操作章节 / 资产 / 镜头 / JSON / 返工

底层 Flow 是执行层
  -> 真实节点
  -> 真实生成任务
  -> 真实资产 URL

两者必须一一映射
  -> 工作台不能是假数据
  -> Flow 不能是另一套平行真相

打开方式补充
  -> 工作台以 fullscreen modal overlay 呈现
  -> 不是浮动面板
  -> 关闭后回到原项目画布
```
