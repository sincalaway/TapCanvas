# 纳米漫剧流水线 Checklist

目标：先补充产品细节，再做 UI，再做数据库设计，最后接智能体与执行链，避免范围失控与关键项遗漏。

## 0. 产品范围确认

- [ ] 明确目标用户是 `单人创作者`、`小团队`、`企业团队` 中的哪一层为首发
- [ ] 明确首发场景是 `AI 漫剧`、`AI 真人剧`，还是统一抽象为 `剧集生产`
- [ ] 明确 MVP 最小闭环：
  - [ ] 项目
  - [ ] 剧集
  - [ ] 剧本/小说导入
  - [ ] 资产拆解
  - [ ] 分镜生成
  - [ ] 镜头出图
  - [ ] 视频生成
  - [ ] 审核
  - [ ] 回改联动
- [ ] 明确哪些能力首发不做：
  - [ ] 企业高级管理
  - [ ] 对外分享门户
  - [ ] 高级空间引擎可视化编辑
  - [ ] 多模型复杂调参面板
