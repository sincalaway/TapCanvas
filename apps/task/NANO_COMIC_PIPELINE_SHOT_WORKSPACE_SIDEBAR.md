# 纳米漫剧流水线镜头工作台右侧栏线框

目标：定义 `episodes/:episodeId/storyboard` 页面右侧栏的固定结构，避免后续把上下文、评论、审核、风险、候选全混成一栏垃圾箱。

- [Dependency Rules](./NANO_COMIC_PIPELINE_DEPENDENCY_RULES.md)
- [Review Flow](./NANO_COMIC_PIPELINE_REVIEW_FLOW.md)

## 1. 右侧栏目标

右侧栏只负责四件事：

1. 展示当前镜头上下文
2. 展示候选与连续性风险
3. 提供审核与评论入口
4. 给出下一步建议

不做：

- 不承担大画布编辑
- 不承担长列表导航
- 不承担系统级诊断

## 2. 栏位结构

建议固定顺序：

```text
┌──────────────────────────────┐
│ 顶部镜头状态摘要             │
├──────────────────────────────┤
│ 上下文卡片                   │
├──────────────────────────────┤
│ 连续性风险卡片               │
├──────────────────────────────┤
│ 候选结果区                   │
├──────────────────────────────┤
│ 审核操作区                   │
├──────────────────────────────┤
│ 评论线程                     │
└──────────────────────────────┘
```

## 3. 顶部镜头状态摘要

必须展示：

- 镜头编号
- 当前镜头标题
- 当前状态
- 是否已审核通过
- 是否有下游视频依赖
- 最近更新时间

示例：

- `镜头 023 · 宫殿正反打`
- `状态：需复检`
- `下游：2 个视频片段`

## 4. 上下文卡片

上下文卡片是“当前镜头为什么长这样”的收口。

建议展示：

- 上一镜摘要
- 当前镜摘要
- 下一镜摘要
- 当前角色
- 当前场景
- 当前固定道具
- 当前风格锁

支持点击查看来源对象：

- 角色卡
- 场景卡
- 风格配置

## 5. 连续性风险卡片

只展示最需要人处理的风险，不展示所有日志。

建议内容：

- 角色一致性风险
- 场景空间风险
- 尾帧承接风险
- 光影连续性风险
- 下游阻塞风险

风险项结构建议：

```text
[风险级别] 角色主设已更新
说明：当前镜头人物外观可能过期
建议：先重绘，再重新提交审核
```

## 6. 候选结果区

作用：看镜头当前有哪些可选结果。

建议能力：

- 当前已选候选
- 其他候选缩略图
- 切换候选
- 标记最佳候选
- 查看生成来源

如果没有候选：

- 显示空状态和推荐动作

## 7. 审核操作区

这里复用审核面板的最小核心操作。

建议按钮：

- `提交审核`
- `通过`
- `打回修改`
- `驳回重做`
- `标记导演确认`

如果镜头处于 `stale/blocked`：

- 顶部必须先给警告
- 审核按钮应根据规则禁用或限制

## 8. 评论线程

评论线程放在最底部，默认保留最近几条。

支持：

- 快速评论
- `@成员`
- 插入风险说明
- 插入审核意见

## 9. 页签式折叠建议

如果右侧信息过多，可拆成 3 个页签：

- `上下文`
- `候选/风险`
- `审核/评论`

但首版更建议单列直排，减少切换成本。

## 10. 状态驱动显示

### 10.1 `draft / ready`

强调：

- 上下文
- 生成入口
- 提交审核

### 10.2 `candidate_ready`

强调：

- 候选比较
- 选择最佳候选
- 连续性风险

### 10.3 `needs_review`

强调：

- 审核操作区
- 历史评论
- 下游影响

### 10.4 `stale / blocked`

强调：

- 风险卡片
- 受影响下游
- 推荐回改动作

## 11. 关键交互

### 11.1 角色改动后

1. 右侧栏顶部显示 `角色主设已更新`
2. 风险卡片显示 `镜头结果可能过期`
3. 候选区保留历史结果但标灰
4. 审核区提示 `当前不能直接通过`

### 11.2 镜头候选切换后

1. 候选区更新选中态
2. 风险卡片刷新
3. 审核状态回到 `needs_review`

### 11.3 审核驳回后

1. 顶部状态更新为 `需返工`
2. 评论区自动插入驳回记录
3. 推荐动作切到 `重绘 / 修改`

## 12. 首版 MVP

首版右侧栏最小实现：

1. 顶部摘要
2. 上下文卡片
3. 风险卡片
4. 候选结果区
5. 审核操作区
6. 评论线程

首版先不做：

- 高级可视化时间轴
- 超复杂连续性评分图
- 多层折叠树
